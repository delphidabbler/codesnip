About Dialog Box

This dialog box provides various pieces of information about CodeSnip and the Code Snippets Database. The dialog contains two tabs:

  • About The Program. This tab shows:
    • The program's copyright information.
    • Copyright and licensing information.
    • Details of where the program's source code can be downloaded.
    • Third party credits.
    • If the program has been registered, and if so, who to.
  • About The Database. This tab shows:
    • Where to find the Code Snippets Database online.
    • A source code disclaimer.
    • A list of contributors to the database along with a list of testers.

The program's version number is displayed in the heading of the dialog box.

If CodeSnip is unregistered then a Register CodeSnip button will be displayed at the bottom left of the dialog box. Clicking this button displays the Registration Wizard that can be used to register the program.

Access this dialog box from the Help | About menu option.
