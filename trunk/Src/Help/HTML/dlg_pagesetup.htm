Page Setup Dialogue Box

This dialogue box is used to specify the properties of a printed page. There are various groups of options:

Paper

  • Use the Size drop down list to specify the size of paper being used.
  • Use the Source drop down list to select the paper source. This box may be disabled if the selected printer does not support it.

Orientation

Use the radio buttons to specify whether the page is to be printed in portrait or landscape format.

Margins

Select the size of each margin by entering a value in the appropriate edit box.

The unit of measurement used to set margins is displayed in the group box label. This can be changed on the Misc. tab of the Preferences dialogue box.

Visual feedback

The image of a page displayed at the top of the dialogue box provides visual feedback about the selected options.
