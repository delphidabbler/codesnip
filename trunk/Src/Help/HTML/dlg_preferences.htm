Preferences Dialogue Box

Use this dialogue box to configure how CodeSnip works. The dialogue box is accessed via the Tools | Preferences menu option.

The dialogue box is in several sections, accessed by tabs across the top of the page. These are:

  • Display
  • Snippet Layout
  • Code Generation
  • Code Formatting
  • Syntax Highlighter
  • Printing
  • News
  • Misc.

Click OK to accept all the changes or Cancel to close the dialogue without making changes.

Note: On some occasions this dialogue box may be displayed with just one tab visible. An example of this is when the Default Options button of the Print dialogue box is clicked – only the Printing tab is displayed.
