Preferences Dialog Box

Use this dialog box to configure how CodeSnip works. The dialog box is accessed via the Tools | Preferences menu option.

The dialog box is in several sections, accessed by tabs across the top of the page. These are:

  • General
  • Code Generation
  • Code Formatting
  • Syntax Highlighter
  • Printing
  • News

Click OK to accept all the changes or Cancel to close the dialog without making changes.

Note: On some occasions this dialog box may be displayed with just one tab visible. An example of this is when the Default Options button of the Print dialog box is clicked – only the Printing tab is displayed.
