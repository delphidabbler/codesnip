Choose Element Colour Dialogue Box

This Windows standard colour dialogue box is displayed when the Custom option is chosen in the Colour drop down list from the Syntax Highlighter page of the Preferences dialogue box.

Use the dialogue box to assign a custom colour to the selected highlighter element. Either choose one of the basic colours or mix you own using the controls on the right of the dialogue box.

Click OK to accept the new colour. This will be displayed as the new custom colour in the drop down list and the colour of the selected highlight element will be updated accordingly. Alternatively click Cancel to abandon your changes.
