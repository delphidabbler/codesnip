Browse Search Path Dialogue Box

This dialogue box is displayed when the ellipsis button to the right of the Add or edit path edit box is clicked in the Configure Compilers dialogue box.

Use this dialogue box to select a directory.

Click OK to confirm the selection. This will close the dialogue box, return to the Configure Compilers dialogue box and enter the selected directory name into the edit box next to the ellipsis button.

Click Cancel to cancel the selection and close the dialogue box without altering the contents of the edit box.
