Detail Pane

This pane appears on the right hand side of the display and shows detailed information about the currently selected snippet or section heading in the Overview pane.

When a snippet is selected the display contains:

  • A brief description of the snippet.
  • The source code (syntax highlighted).
  • A list of standard Delphi units required by the snippet.
  • A list of other snippet called by the selected snippet.
  • A cross reference to other similar snippets, labelled "See also".
  • A table indicating which compilers the snippet is compatible with. (Not displayed for freeform snippets.)
  • Optionally, some further information about the snippet, including credits and links to related web sites.

Clicking any of the snippets in the "Required snippets" and "See also" lists causes the snippet in question to be displayed. You can return to the original snippet using the Previous toolbar button.

If a section header is selected in the Overview Pane, a list of all the snippets belonging to the section is displayed. A brief description of each snippet is also displayed.

Related Actions

  • Save a selected snippet to disk by using the Save Snippet dialog box that is accessed from the Save Snippet toolbar button. (Not available for freeform snippets).
  • Copy the selected snippet to the clipboard using one of the options on the Edit menu.
