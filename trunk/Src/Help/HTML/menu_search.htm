Search Menu

The Search menu contains the following options:

Find Text
[Ctrl+F] 		Finds snippets that contain specified text. The Find Text dialogue box is displayed where the required text is specified along with other search criteria. On completion of a successful search the Overview Pane shows only the snippets that match the search criteria.
Find Compiler
[Ctrl+Shift+F] 		Finds snippets that either compile, fail to compile or result in warnings when compiled with specified compilers. Displays the Find Compiler dialogue box where the search is defined. On completion of a successful search the Overview Pane shows only the snippets that match the search criteria.
Find Cross Refs
[Ctrl+R] 		Finds snippets cross referenced by the currently selected snippets. Displays the Find Cross References dialogue box where the search is customised. On completion of a successful search the Overview Pane shows only the snippets that match the search criteria.
Select Snippets
[Shift+Ctrl+S] 		Allows manual selection of the snippets to be displayed. Displays the Select Snippets dialogue box where the required snippets are selected. On completion the Overview Pane shows only the selected snippets.
Show All
[Ctrl+Shift+A] 		Restores the whole database after a find operation. All snippets in the database are listed in the Overview Pane. This menu option is only enabled after a successful find operation.
