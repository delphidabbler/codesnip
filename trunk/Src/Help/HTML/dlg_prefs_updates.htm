Updates Preferences

Use this tab of the Preferences dialogue box to determine how CodeSnip checks for updates. The tab is accessed by selecting the Tools | Preferences menu item then clicking the Updates tab.

There are two check boxes:

  • Check for program updates on start-up
    Tick this box if you want CodeSnip to check online for any available updates to the program. If you clear the box CodeSnip will not perform the checks. You can still do this manually using the Tools | Check For Program Updates menu option.
  • Check for database updates on start-up
    When this box is ticked CodeSnip will check online to find out if any updates to the Code Snippets Database are available for download. If your clear the check box CodeSnip will not do this, but you can still do it manually from the Database | Update From Web menu option.

CodeSnip checks for updates in a background thread shortly after starting up. If there are no updates available you will see nothing. If any updates are available a small notification window is displayed in the bottom right hand side of the Detail Pane. The notification window will be displayed for about 20 seconds or until you close it.

The notification window will describe the available update and provide a button to click to enable it to be downloaded. There will also be a check box you can use to inhibit the display of further notifications. This has the same effect as clearing the related check box in this preferences tab.
