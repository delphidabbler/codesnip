Navigating the Database

There are numerous ways to navigate the database.

Overview Pane

The Overview Pane displays snippets by name, organised into groups.

Clicking a snippet's name displays the snippet in the Detail Pane.

Clicking a group heading displays a table of contents of the group in the Detail Pane. This group contains all the snippets from the group along with their descriptions.

Selected items are normally† displayed in the current tab of the Detail Pane. However, Ctrl clicking any item displays it in a new tab†.

Details Pane

The Details Pane is often displaying either details of a snippet or a group's table of contents.

A snippet display may contain links to related snippets and to the category to which it belongs.

A group's table of contents contains links to the snippets contained in the group.

Clicking any link displays target of the link in the current tab†, overwriting the current display. Once again Ctrl clicking the link displays the target item in a new tab†.

You can use the Tab key to cycle through any snippet and category links displayed in the Details Pane and then press Return or Ctrl+Return to display the links in current or a new tab, respectively†.

History

Previously viewed items (i.e. tables of contents and snippets) can be re-displayed using the navigation history.

You can move backwards and forwards in the history  Previous and  Next toolbar buttons or the equivalent View menu items.

Alternatively you can jump directly to any item in the history using the drop-down menus next to the Previous and Next buttons.

Ctrl clicking on the history buttons and menu items works as you would expect: it displays the selected snippet in a new tab†.

Searching

You can narrow down the range of displayed snippets by performing a ssearch.

Footnote

† If an item is already displayed in a Detail Pane tab and it is reselected then the existing tab will be reselected, and updated if necessary, regardless of whether the link was clicked or Ctrl clicked. This also applies to the use of Return and Ctrl+Return on active links.
