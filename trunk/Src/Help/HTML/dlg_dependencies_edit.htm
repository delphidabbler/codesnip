Snippets Editor: Dependencies Dialogue Box

This dialogue box is displayed when the View Dependencies option is selected from the context menu of the Dependencies list box on the References tab of the Snippets Editor.

If the snippet being edited has dependenciess selected in the Dependencies list box then a tree of those snippets is displayed that shows both the direct and indirect dependencies. When there are no dependencies a message is displayed to that effect.

If a circular reference is detected (i.e. the snippet depends directly or indirectly upon itself) the circular reference is displayed in red and an error message is displayed. Circular references are not permitted and must be corrected.
