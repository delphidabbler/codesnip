Export File Dialogue Box

This Windows standard dialogue box is displayed when the ellipsis button to the right of the Save to file edit box is clicked in the Export Snippets dialogue box.

Use this dialogue box to specify the name of the export file.

Click OK to confirm the selection. This will close the dialogue box, return to the Export Snippets dialogue box and enter the selected file name into the edit box next to the ellipsis button.

Click Cancel to cancel the selection and close the dialogue box without altering the contents of the edit box.
