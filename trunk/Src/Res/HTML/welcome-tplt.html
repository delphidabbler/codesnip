Welcome to CodeSnip by DelphiDabbler

CodeSnip maintains a repository of your source code snippets. It can also download and display the Object Pascal snippets stored in the online DelphiDabbler CodeSnip Database.

You have not yet created any of your own snippets. Use the Snippets | New Snippet menu option to get started. More info.

Overview

Code snippets are listed in the Overview pane on the left hand side of the window. You can display the snippets either by category, alphabetically or by type of snippet by using the pane's tabs. Colours differentiate snippets from the online database and the local repository.

Selecting a snippet's name in the Overview pane causes the a description of the snippet, its source code and other information to be displayed in a tab in the Information Tab of the Details pane. The information displayed can be customised using the Snippet Layout tab of the Preferences dialogue box.

You can have more than one tab open in the Details pane. Open a new tab using the View | New Tab menu option or use the Ctrl+T shortcut. Use the View | Close Tab menu option or press Ctrl+F4 to close a tab. You can also display a snippet directly in a new tab by Ctrl clicking it in the Overview pane.

Pascal snippets can be test compiled using any Delphi or Free Pascal compiler available on your computer. Before you can do this you need to tell CodeSnip where to find the compilers by using the Configure Compilers dialogue box .
