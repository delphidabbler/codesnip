Welcome to CodeSnip by DelphiDabbler

CodeSnip maintains a repository of your source code snippets. It can also access and display the Object Pascal routines, types and constants stored in the online DelphiDabbler CodeSnip Database.

You have not yet created your own repository. Use the Database menu to do this. More info.

You do not have a copy of the online database. Download it.

Overview

Code snippets are listed in the Overview pane on the left hand side of the window. You can display the snippets either by category, alphabetically or by type of snippet by using the pane's tabs. Colours differentiate snippets from the online database and the local repository.

Selecting a snippet's name in the Overview pane causes the a description of the snippet, its source code and other information to be displayed in the Information Tab of the Details pane.

The Detail pane's Compiler Check Tab is used to test compile a selected snippet in any supported Pascal compiler that is installed on your computer. Use the Configure Compilers dialogue box to tell CodeSnip about the available compilers.
