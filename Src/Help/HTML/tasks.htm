Common Tasks

There are several tasks that are commonly performed when using CodeSnip. They are:

  • Add Snippets to the Database
  • Save a Code Snippet to Disk
  • Copy a Code Snippet to Clipboard
  • Print Information about a Snippet
  • Generate a Pascal Unit
  • Search the Code Snippets Database
  • Test Compile a Snippet
  • Update the Database
  • Customise CodeSnip
  • Submit Code to the Database
  • Export and Import Snippets
