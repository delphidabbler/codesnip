User Defined Highlighters Dialogue Box

This dialogue box is displayed from the User Defined Styles option of the drop-down menu displayed by the Use Named Style button located on the Syntax Highlighter tab of the Preferences dialogue box.

The dialogue box is used to manage and display saved user-defined syntax highlighter styles. The available styles are listed in the Available highlighter styles list box.

To display a style either select the required style and press the Use & Close button or, alternatively, double click the style in the list box. The dialogue box will be closed immediately and the style will be displayed in the main dialogue box ready for editing.

You can also delete unwanted styles from the dialogue box by selecting the required style name and then clicking the Delete button. You will be asked to confirm this action. Note: the deleted style will be restored if you cancel the Preferences dialogue box instead of OKing it.

The dialogue box can be closed without selecting one of the listed styles by clicking the Done button.
