View Menu

The View menu contains the following options:

Previous
[Alt+Left] 		Shows the previously displayed page (snippet, category etc.) in the Detail Pane.
Next
[Alt+Right] 		Displays the next page in the Detail Pane. Only available if Previous has been selected at least once.
Welcome Page
[Alt+Home] 		Redisplays the "welcome page" that is shown at start-up.
  Categorised Snippets Displays snippets grouped by category in the Overview Pane.
  Alphabetical Snippets Displays snippets in alphabetical order, grouped by initial letter in the Overview Pane.
  Snippets by Kind Displays snippets grouped by kind in the Overview Pane: i.e. routine, type, constant or freeform.
New Tab
[Ctrl+T] 		Opens a new empty tab in the Detail Pane.
Close Tab
[Ctrl+F4] or [Ctrl+W] 		Closes the current selected tab, if any, in the Detail Pane.
  Close All Tabs Closes all currently open tabs in the Detail Pane.
Dependencies
[Ctrl+D] 		Displays a tabbed dialogue box that recursively shows all the snippets on which the selected snippet depends, along with all the snippets that directly depend on it.
Collapse Section
[Ctrl+Num Sub] 		Collapses the selected section in the Overview Pane if possible.
Expand Section
[Ctrl+Num Plus] 		Expands the selected section in the Overview Pane if possible.
Collapse All Sections
[Shift+Ctrl+Num Sub] 		Collapses all sections in the Overview Pane so that only the section headings a visible.
Expands All Sections
[Shift+Ctrl+Num Plus] 		Expands the all sections in the Overview Pane so that all snippets are visible.
