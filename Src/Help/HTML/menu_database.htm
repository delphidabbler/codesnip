Database Menu

The Database menu contains the following options:

Save User Database
[Ctrl+S] 		Saves the user defined database. Not available if the database has not changed.
Backup User Database Creates a backup of the user defined database. The backup is stored in a file specified by the user via the Save Backup dialogue box.
Restore User Database Restores the local database from a backup. The backup file to be restored is chosen by the user via the Open Backup File dialogue box.
  Install or Update DelphiDabbler Snippets Database Displays the Install or Update DelphiDabbler Snippets Database dialogue box that enables the main code snippets database to be updated from a local copy of the DelphiDabbler Snippets Database.
  Move User Database† Displays the Move User Database dialogue box from where the user defined database files can be moved to a diffent folder. The main reason for this option is so that the user can locate the database in a suitable location for backing up, such as a DropBox or Google Drive sub-directory.

† The Move User Database option is not available in the portable edition of CodeSnip.
