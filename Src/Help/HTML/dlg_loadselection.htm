Load Selection Dialogue Box

This Windows standard dialogue box is displayed when the File | Load Selection menu option is chosen.

Use this dialogue box to specify the name of the file containing details of the database selection to be loaded. By default only files with the ".cssel" extension are displayed. This can be changed using the Files of type drop-down list and chosing "All files".

Click OK to load the selection. Choosing an invalid file will result in an error message.
