Export File Dialog Box

This Windows standard dialog box is displayed when the ellipsis button to the left of the Save to file edit box is clicked in the Export Snippets dialog box.

Use this dialog box to specify the name of the export file.

Click OK to confirm the selection. This will close the dialog box, return the the Export Snippets dialog box and enter the selected file name into the edit box next to the ellipsis button.

Click the Cancel to cancel the selection and close the dialog box without altering the contents of the edit box.
