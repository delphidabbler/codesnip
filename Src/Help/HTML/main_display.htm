The Main Display

CodeSnip's main window is divided into four parts:

  1. Menu menu
    The main menu appears at the top of the window.
  2. Tool bar
    The tool bar sits just below the main menu. It has tool buttons that provide short cuts to a subset of menu commands.
  3. Overview pane
    The Overview pane is on the left of the window. It is used to list all the snippets in the database. Snippets are organised by category, alphabetically or by snippet kind.
  4. Detail pane
    This pane is on the right hand side of the window. It displays detailed information about selected snippets and other items of information. Information is displayed in one or more tabs.
  5. Status bar
    The status bar lies across the bottom of the window. It displays information about the database and any actives searches. When a main or context menu item is highlighted brief help text describing the menu item is also displayed here.

The relative sizes of the Overview and Detail panes can be changed by dragging the vertical splitter bar that separates the two panes.
