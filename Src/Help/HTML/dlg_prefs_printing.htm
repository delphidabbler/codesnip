Printing Preferences

This tab of the Preferences dialogue box is used customise the default appearance of printed output.

The tab is accessed by selecting the Tools | Preferences menu item then clicking the Printing tab. You can also display this tab from the Print dialogue box.

Warning: Options set in this tab usually take effect the next time the application is started. The exception is when the tab is displayed from the Print dialogue box, when you can choose to apply the new options straight away.

Document Formatting Options

There are two options that change the way printed documents are formatted:

  • The Syntax highlight source code check box specifies whether source code included in the output is syntax highlighted by default. Plain mono-spaced text is used when the check box is cleared.
  • The Use colour check box indicates whether the document is produced in colour or in black and white.

A preview of the effect of the options is displayed to the right of the check boxes.

When syntax highlighting is enabled the style used is as specified in the Syntax Highlighter tab. When black and white printing is specified the highlighter's font styles are used but colours are ignored.

Page Margins

The default page margins can be specified using the Left, Right, Bottom and Top edit boxes.

Margins are displayed and entered either in inches or millimeters. The units being used are displayed in the group box caption. Note: measurement units can be changed on the Misc. tab, if it is visible.
