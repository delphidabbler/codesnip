Compile Menu

The Compile menu contains the following options:

Test Compile Snippet
[F9] 		Attempts to compile the currently selected snippet with each available compiler and displays the results in the Test Compile Results dialogue box. See About Compiler Checks for more information. This option is disabled if the snippet is freeform.
View Compile Errors Displays any test compilation errors and warnings in a dialogue box. This option is disabled if the snippet is freeform.
View Test Unit Displays a dialogue box that shows source code of the unit used to test compile the selected snippet. This option is disabled if the snippet is freeform.
