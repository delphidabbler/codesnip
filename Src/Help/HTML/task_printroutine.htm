Print Snippet Information

You can print information about a snippet as follows.

  1. Select a snippet in the Overview Pane.
  2. Click the toolbar button or choose the File | Print menu option.
  3. Select a printer and the required print options in the Print dialogue box. You can also click the Page Setup button to set up the page size, orientation and margins from the Page Setup dialogue box.
  4. Click OK to print the document to the selected printer.

The printed document contains the same information that is presented in the detail pane.

Note that the default output options and margin size can be configured on the Printing page of the Preferences dialogue box.
