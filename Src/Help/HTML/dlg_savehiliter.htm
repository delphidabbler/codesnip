Save Highlighter Dialogue Box

This dialogue box is displayed by clicking the Save Style button in the Syntax Highlighter page of the Preferences dialogue box.

The dialogue box is used to save a custom syntax highlighter style. Provide a name for the style in the Save current highlighter as combo box. You can overwrite any existing style by selecting its name from the combo box's drop down list.

Style names must be 32 characters or less.

You can use a named style by selecting it in the User Defined Highlighters dialogue box, accessed from the Syntax Highlighter preferences page's Use Named Style drop-down button.
