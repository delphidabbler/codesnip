Snippets Menu

The Snippets menu contains the following options:

New Snippet
[Ctrl+Ins] 		Displays the Add a Snippet dialogue box to enable the user to enter a new user-defined snippet.
Edit Snippet
[Ctrl+F2] 		Edits the currently selected user defined snippet using the Edit Snippet dialogue box.
Duplicate Snippet
[Shift+Ctrl+D] 		Duplicates the currently selected snippet as a new user-defined snippet. The new snippet's name and category are specified in the Duplicate Snippet dialogue box.
Delete Snippet
[Ctrl+Del] 		Deletes the currently selected user defined snippet.
  Export Snippets Exports selected user defined snippets for importing into another user's user defined database. The Export Snippets Dialogue Box is displayed where the snippets and export file name are specified.
  Import Snippets Imports user defined snippets into the user database. The Import Wizard is displayed which permits the import to be customised.
  Import Snippets From SWAG Imports selected snippets from the SWAG Database into the local user-defined snippets database. The SWAG Import Wizard is displayed which is used to select the required snippets.
Add To Favourites
[Shift+Alt+A] 		Makes the currently display snippet a "Favourite". If the snippet is already a favourite this option has no effect.
  Show Favourites
[F4] 		Displays the Favourites non-modal dialogue box. If the dialogue box is already displayed it is activated and given the focus.
