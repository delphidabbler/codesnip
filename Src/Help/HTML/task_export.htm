Exporting and Importing Snippets

You can export user defined snippets from your database in a format you can send to other CodeSnip users who can then import them into their user-defined database.

Use the Snippets | Export Snippets menu option to create an export file and use the Snippets | Import Snippets menu item to import a file someone else has sent you.

Imported snippets will be located in the user-defined "Imported Snippets" category. They can be moved to other categories by opening each snippet in the snippets editor and changing its category there.

If you have a separate snippets databases maintained by different major versions of CodeSnip, say by v3 and v4, you can export snippets from the older version and import the files into the newer version, but not the reverse.
