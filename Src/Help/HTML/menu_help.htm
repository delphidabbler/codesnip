Help Menu

The Help menu contains the following options:

Contents
[F1] 		Displays the help contents tab of this help file along with the Overview topic.
  QuickStart Guide Displays the QuickStart topic of this help file. Provides a brief overview of CodeSnip to get you up and running quickly.
  About Compiler Checks Displays the About Compiler Checks topic of this help file. This topic explains how to use CodeSnip's ability to test compile snippets.
License Selecting this option displays a summary of CodeSnip's end user license agreement in the help file.
CodeSnip On GitHub Displays the default page of CodeSnip's GitHub Project in the default web browser. The latest release of CodeSnip is always published here.
CodeSnip News Blog Displays the CodeSnip Blog in the default web browser. The latest news about CodeSnip is posted in the blog.
FAQs Displays the CodeSnip FAQs in the default web browser.
About Displays the about box that shows the program's version number, copyright information and credits. It also provides links to the full license for the program. See if you can find the Easter Egg!
