Page Setup Dialog Box

This dialog box is used to specify the properties of a printed page. There are various groups of options that are described below.

Paper

  • Use the Size drop down list to specify the size of paper being used.
  • Use the Source drop down list to select the paper source. This box may be disabled if the selected printer does not support it.

Note: this setting is not persistent and must be set each time you print.

Orientation

Use the radio buttons to specify whether the page is to be printed in portrait or landscape format.

Note: this setting is not persistent and must be set each time you print.

Margins

Select the size of each margin by entering a value in the appropriate edit box.

The unit of measurement used to set margins is displayed in the group box label. This can be changed on the General tab of the Preferences dialog box.

Visual feedback

The image of a page displayed at the top of the dialog box provides visual feedback about the selected options.
