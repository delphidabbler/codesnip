Tools Menu

The Tools menu contains the following options:

Preferences Displays the Preferences dialogue box where the user can configure various aspects of the application.
Configure Compilers Displays the Configure Compilers dialogue box where the compilers used in test compilations are specified.
  Proxy Server Enables CodeSnip to be configured to use a proxy server when accessing the internet. Details of the proxy are entered in the Proxy Server Configuration dialogue box that is displayed.
  Check For Program Updates Checks if an updated version of CodeSnip is available. The Check For Program Updates dialogue box is displayed which checks online for any updates and reports if one is available.
Report Bug Online Displays the Bug Report Dialogue Box. This is used to access the CodeSnip bug tracker.
