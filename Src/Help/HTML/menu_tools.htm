Tools Menu

The Tools menu contains the following options:

Preferences Displays the Preferences dialog box where the user can configure various aspects of the application.
Configure Compilers Displays the Configure Compilers dialog box where the compilers used in test compilations are specified.
  Proxy Server Enables CodeSnip to be configured to use a proxy server when accessing the internet. Details of the proxy are entered in the Proxy Server Configuration dialog box that is displayed.
Report Bug Online Displays the Bug Report Dialog Box. This is used to access the CodeSnip bug tracker.
  Register CodeSnip Displays the Registration Wizard that enables the program to be registered online. Registration is not compulsory, but is appreciated.
Note: This menu item only appears if CodeSnip has not been registered.
