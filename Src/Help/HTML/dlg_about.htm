About Dialog Box

This dialog box provides various pieces of information about CodeSnip and the Code Snippets Database. It is accessed from the Help | About menu option.

The program's version number is displayed in the heading of the dialog box.

If CodeSnip is unregistered then a Register CodeSnip button will be displayed at the bottom left of the dialog box. Clicking this button displays the Registration Wizard that can be used to register the program.

In addition the dialog has three tabs, each of which is described below.

About The Program

This tab shows:

  • The program's copyright information.
  • Copyright and licensing information.
  • Details of where the program's source code can be downloaded.
  • Third party credits.
  • If the program has been registered, and if so, who to.

About The Database

This tab shows:

  • Where to find the Code Snippets Database online.
  • A source code disclaimer.
  • A list of contributors to the database along with a list of testers.

Paths

This tab gives information about the various directories used by CodeSnip. They are:

  • Install Directory – the directory where CodeSnip is installed, or more accurately, the directory from which the current instance of the program was executed.
  • Main Database Directory – the directory containing a copy of the main, online, database. This directory may be empty if the user has not downloaded the online database.
  • User Database Directory – the directory containing the user defined database for the current user. This directory may be empty if the user has not created any user defined snippets.

The full path to any directory may be shortened if there is no enough room to displayed it.

Clicking the button to the right of each directory will open that directory in Windows Explorer.
