Delete Category Dialog Box

This dialog is used to delete a user-defined category from the database. It is accessed from the Database | Delete Category menu option.

Warning: Deleting a category that contains snippets also deletes all those snippets from the database. The dialog box will warn you if this is about to happen.

Select the category to be deleted from the list box. then click Delete to delete it. If the category contains any snippets a dialog box will be displayed that asks you to confirm you wish to delete the category and all its snippets. Click Yes to confirm.

The Delete button will be disabled if you haven't selected a category from the list box.

You may have noticed that some user defined categories do not appear in the list of categories to be deleted. This is because some categories are required by CodeSnip and cannot be deleted, although they can be renamed.
