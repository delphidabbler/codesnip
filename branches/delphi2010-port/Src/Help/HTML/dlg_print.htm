Print Dialog Box

This dialog box is displayed when you choose to print information about a snippet using the File | Print menu option. The printed output contains the same information that is displayed in the Information pane.

Use this dialog box to:

  • Select the required printer from the Installed Printers drop down menu.
  • View and edit the page options using the Page Setup dialog box. Click the Page Setup button to view this dialog box.
  • Customise the format of the printed document using the controls in the Document Formatting Options group box as follows:
    • Check the Syntax highlight source code check box to syntax highlight source code contained in the document. Plain mono-spaced text is used when the check box is cleared.
    • The Use colour check box indicates whether the document is produced in colour (checked) or in black and white (unchecked). Note that some printers can be configured to produce grey scale documents regardless of this setting.

Clicking OK prints the document while Cancel aborts the print. Printing cannot be aborted once OK is clicked.

The Preferences button, to the left of the OK button, displays the Printing page of the Preferences dialog box where the default document formatting options and page margins can be configured.
