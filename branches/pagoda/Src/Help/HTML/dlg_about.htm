About Dialogue Box

This dialogue box is accessed from the Help | About menu option and displays information about CodeSnip.

The program's version number is displayed in the heading of the dialogue box.

In addition the dialogue has two tabs, each of which is described below.

1. About The Program

This tab shows:

  • The program's copyright and licensing information.
  • Credits and Attributions.

2. Paths

This tab gives information about various directories used by CodeSnip. They are:

  • Install Directory – the directory where CodeSnip is installed or, more accurately, the directory from which the current instance of the program was executed.
  • User Database Directory – the directory containing the current user's database.

The full path of any directory may be shortened if there is not enough room to display it all. In this case a pop-up window will display the path in full if the mouse cursor is hovered over the text.

Clicking the button to the right of each directory will open that directory in Windows Explorer. If the directory does not exist then the button will be disabled.
