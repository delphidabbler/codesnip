Save Snippet to Disk

You can save either a single routine, or all the routines associated with a given tag, to disk as follows.

  1. Select a snippet or a tag in the Overview Pane.
  2. Click the toolbar button or choose the File | Save Annotated Source menu option.
  3. Enter the name of the snippet file and choose the file type and other options in the Save Annotated Source dialogue box.
  4. Choose the file type, the commenting style and whether you want the file syntax highlighted then click OK to save the file.

Note that only routine snippets and routine snippets associated with a given tag can be saved (non-routine snippets with the required tag are ignored).

Note that the default commenting style, syntax highlighting and file type can be configured using the Preferences dialogue box.
