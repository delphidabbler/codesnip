Overview Pane

This pane appears on the left hand side of the main window. It lists all the snippets in the database, grouped under section headers, which appear in bold. Snippets can be grouped in three ways:

  1. by category, using the Categorised tab;
  2. alphabetically, using the Alphabetical tab;
  3. by snippet kind using the Snippet Kind tab.

If a search has been run, the snippets that are displayed are just those that match the search.

Section headers for sections that contain no snippets are hidden by default. This can be changed using the Display tab of the Preferences dialogue box.

Choosing an item in the overview pane causes detailed information about it to appear in the Details Pane. The selected item normally appears in the current tab in the details pane. However, if the item is clicked whilst holding down the Ctrl key, the item is displayed in a new tab in the detail pane.

The nodes of the tree used to display the routines can be expanded and collapsed in several ways:

  • Double clicking a section header toggles it between expanded and collapsed states.
  • Clicking the arrow to the left of a section header toggles it between expanded and collapsed states.
  • Clicking the button in the title of the pane fully expands the tree while clicking the button fully collapses the tree.
  • Menu items on the View menu and the overview pane's context menu can also be used to expand and collapse sections.
  • Finally some key presses will expand and collapse sections: these are listed below.

You can configure whether the overview pane tree view appears fully expanded or fully collapsed by using the Display tab of the Preferences dialogue box.

Rolling the mouse over a tree item highlights it, and if it's text is not completely visible, displays a tool-tip showing the complete text.

Keyboard Short-cuts

The following keyboard short-cuts can be used when the overview pane has focus.

Up Moves to the previous entry in the tree, if any, and displays it.
Down Moves to the next entry in the tree, if any, and displays it.
Shift+Up Selects the section header above the currently selected item and displays the section's details in the details pane. No action is taken if the first section header is already selected.
Shift+Down Selects the section header below the selected item and displays the section's details in the details pane. If the selected item is part of the last section, its own section header is selected.
Ctrl+Up Scrolls up the tree, without changing the selected item.
Ctrl+Down Scrolls down the tree, without changing the selected item.
Left
  • If a sub item is currently selected, the item's parent node is selected.
  • If an expanded section head item is currently selected, it's section is collapsed.
  • If a collapsed section head is currently selected, no action is taken.
Right
  • If a collapsed section header is selected the section is expanded.
  • If an expanded section header is selected the section's first child node is selected.
  • If a child node or an empty section header is selected no action is taken.
Ctrl+Left Pans to the left of the tree, without changing the selected item.
Ctrl+Right Pans to the right of the tree, without changing the selected item.
Page Up Jumps up one page in the tree and displays the newly selected item.
Page Down Jumps down one page in the tree and displays the newly selected item.
Home
  or
Ctrl+Home 		Selects and displays the first item in the tree.
End
  or
Ctrl+End 		Selects and displays the last item in the tree.
Return
  • When a section header is selected it is toggled between expanded and collapsed state.
  • When a child node is selected no action is taken.
Ctrl+Num Sub Collapses an expanded section and selects the section header.
Ctrl+Num Plus Expands a collapsed section.
Shift+Ctrl+Num Sub Collapses all sections in the tree.
Shift+Ctrl+Num Plus Fully expands the tree.
