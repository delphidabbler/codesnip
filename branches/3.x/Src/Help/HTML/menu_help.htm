Help Menu

The Help menu contains the following options:

Contents (F1) Displays the help contents tab of this help file along with the Welcome topic.
  QuickStart Guide Displays the QuickStart topic of this help file. Provides a brief overview of CodeSnip to get you up and running quickly.
  About Compiler Checks Displays the About Compiler Checks topic of this help file. This topic explains how to use CodeSnip's ability to test compile snippets.
  Privacy Statement Selecting this option displays CodeSnip's privacy statement in the help file.
License Selecting this option displays CodeSnip's end user license agreement in the help file.
On The Web This item leads to a sub menu. See below for information.
  CodeSnip News Displays the CodeSnip News dialog box that shows the latest news about CodeSnip and the online database.
Donate Choose this option to make a donation to help support development of CodeSnip and the online database. Displays a dialog box containing a button to click to make a payment online using PayPal™.
About Displays the about box that shows the program's version number, copyright information and credits. It also provides links to the full license for the program and shows if the program has been registered. See if you can find the Easter Egg!

The On The Web sub menu contains the following items:

  CodeSnip Home Page Displays the CodeSnip Home Page in the default web browser. The latest release of CodeSnip is always published here first.
  FAQs Displays the CodeSnip FAQs in the default web browser.
  DelphiDabbler Home Page Displays the home page of the DelphiDabbler web site in the default web browser. The site has more open source and free software, open source Delphi units, components and IDE extensions. Several articles on various programming subjects are also available.
