Save Selection Dialogue Box

This Windows standard dialogue box is displayed when the File | Save Selection menu option is chosen.

Use this dialogue box to specify the name of the file that is to store details of the current database selection. By default only files with the ".cssel" extension are displayed. This can be changed using the Save as type drop-down list and chosing "All files".

Click OK to confirm the selection. If the file aleady exists you will be prompted for permission to overwrite it. Note that if the "CodeSnip selection files" file type is selected and a file name with no extension is entered then ".cssel" will be appended to it.
