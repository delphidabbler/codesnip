Miscellaneous Preferences

This tab of the Preferences dialogue box is used for miscellaneous preferences that don't merit a tab of their own. The tab is accessed by selecting the Tools | Preferences menu item then clicking the Misc. tab.

There is only a single option that can be set on this tab.

Measurement

Select the measurement units to be used by the application from the Preferred units of measurement drop down list. Options are Inches and Millimetres. If you have not set this option before, the measurement units will default to your locale settings.
