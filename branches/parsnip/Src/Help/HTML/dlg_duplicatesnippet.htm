Duplicate Snippet Dialogue Box

This dialogue box is used to duplicate the currently selected snippet, creating a new user defined snippet. The dialogue box is accessed from the Snippet | Duplicate Snippet menu option.

Enter a title for the duplicated snippet in the Title edit control. The current snippet's title will be displayed in the control. You may use the same title for the duplicated snippet but it is best to change it to make it easier to distinguish between different snippets.

The newly duplicated snippet can be edited in the Snippets Editor immediately after clicking the OK button. To do this tick the Edit in Snippets Editor check box. If you don't want to display the Snippets Editor leave the check box clear.

Click OK to create the duplicate snippet or click Cancel to abandon the process.

Note: Upon clicking OK the duplicated snippet is added to the database immediately. If you have chosen to open the Snippet Editor automatically, this is done after the snippet has been added to the database. Therefore cancelling the Snippets Editor will not remove the duplicated snippet from the database: it merely cancels any changes you made in the editor. To remove the snippet select it and choose the Snippet | Delete Snippet menu option or press Ctrl+Del.
