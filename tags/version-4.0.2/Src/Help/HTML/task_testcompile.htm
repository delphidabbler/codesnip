Test Compile a Snippet

You can check any snippet in the database with all the supported compilers installed on the local computer, providing the snippet is not freeform. Proceed as follows:

  1. Select a snippet in the Overview Pane.
  2. Click the Compile | Test Compile Snippet menu option or press F9.

The Test Compile Results dialogue box will appear and display the compilation results. There may be a pause before the results are displayed while the compilation is performed.
