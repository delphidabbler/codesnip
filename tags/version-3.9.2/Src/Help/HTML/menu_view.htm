View Menu

The View menu contains the following options:

Previous (Alt+Left) Shows the previously displayed page (snippet, category etc.) in the Detail Pane.
Next (Alt+Right) Displays the next page in the Detail Pane. Only available if Previous has been selected at least once.
Welcome Page (Alt+Home) Redisplays the "welcome page" that is shown at start-up.
  Categorised Snippets Displays snippets grouped by category in the Overview Pane.
  Alphabetical Snippets Displays snippets in alphabetical order, grouped by initial letter in the Overview Pane.
  Snippets by Kind Displays snippets grouped by kind in the Overview Pane: i.e. routine, type, constant or freeform.
  Detailed Information Displays the Information Tab in the Detail Pane. This tab shows information about the selected snippet or category.
  Compiler Check Shows the Compiler Checks Tab within the Detail Pane. This page enables a selected snippet to be test compiled in all available compilers.
Test Unit Displays a dialog box that shows source code of the unit used to test compile the selected snippet. This option is not available if the snippet is freeform.
Dependencies (Ctrl+D) Displays a dialog box that recursively shows all the snippets on which the selected snippet depends.
Collapse Tree Node (Ctrl+Num Sub) Collapses the selected tree node in the Overview Pane if possible.
Expand Tree Node (Ctrl+Num Plus) Expands the selected tree node in the Overview Pane if possible.
  Collapse All (Shift+Ctrl+Num Sub) Collapses the whole tree in the Overview Pane so that only the section headings a visible.
  Expands All (Shift+Ctrl+Num Plus) Expands the whole tree in the Overview Pane.
