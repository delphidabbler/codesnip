Choose Element Colour Dialog Box

This Windows standard colour dialog box is displayed when the Custom option is chosen in the Colour drop down list from the Syntax Highlighter page of the Preferences dialog box.

Use the dialog box to assign a custom colour to the selected highlighter element. Either choose one of the basic colours or mix you own using the controls on the right of the dialog box.

Click OK to accept the new colour. This will be displayed as the new custom colour in the drop down list and the colour of the selected highlight element will be updated accordingly. Alternatively click Cancel to abandon your changes.
