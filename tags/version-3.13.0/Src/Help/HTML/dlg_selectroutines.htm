Select Snippets Dialog Box

Use this dialog to manually select the snippets that are displayed. The dialag is displayed from the Search | Select Snippets menu option.

Snippets are selected simply by placing a check mark next to the name of the required snippets. Checking or unchecking a category will select or de-select all the snippets in the category. When the dialog box appears all snippets currently displayed by the program are selected.

There are also four buttons that can be used to select or deselect groups of snippets. The options are:

  • Select All selects all the snippets in the database.
  • Clear All deselects all the snippets in the database.
  • User Defined selects all the user defined snippets in the database. The button will be disabled if there are no user defined snippets.
  • Main selects all the snippets in the main database. If there are no user defined snippets Main has the same effect as Select All.

When the selection has been made, click the OK to accept the changes. At least one snippet must be selected. Click Cancel to abort your selection.

The main purpose of the Select Snippets dialog box is to enable you to specify precisely which snippets will be included in a generated unit. See the Generate Unit task for information about this subject.
