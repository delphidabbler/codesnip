Detail Pane

This pane appears on the right hand side of the display and shows detailed information about the current selection in the Overview pane.

The detail pane contains two tabs, each of which display different information relating to the selection:

  • The Information tab shows detailed information about the selected snippet or section.
  • The Compiler Check tab can test-compile a selected snippet in any supported compiler that is installed on the local computer.

Both tabs can also display the Welcome Page.

Related Actions

  • Save the selected snippet to disk by using the Save Snippet dialog box that is accessed from the Save Snippet toolbar button. (Not available for freeform snippets).
  • Copy the selected snippet to the clipboard using one of the options on the Edit menu.
