Navigating the Database

The database is navigated in one of several ways:

  • By clicking on a snippet name in the Overview Pane. This displays the snippet in the Detail Pane.
  • By clicking a section heading in the Overview Pane. This displays a list of snippets in the Detail Pane. Clicking the name of any snippet in the list displays it.
  • Clicking links to snippets in a "See also" and "Required snippets" list in a snippet's detailed display.
  • Using the Previous and Next toolbar buttons or equivalent View menu items to move backwards and forwards through recently visited pages. You can use the drop-down lists next to the buttons to jump directly to pages.

You can narrow down the range of displayed routines by performing a Search.
