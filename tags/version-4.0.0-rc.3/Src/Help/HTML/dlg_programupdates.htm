Check For Program Updates Dialogue Box

This dialogue box is displayed when the Tools | Check For Program Updates menu option is selected.

As soon as it appears it contents the DelphiDabbler web site to check to see if any new versions of CodeSnip are available. The message "Checking..." is displayed while this is in progress.

If there is a new version then a button is displayed that, when clicked, displays CodeSnip's download web page in your default web browser. You can download the latest version from there.

If there is no new version, a message is displayed to that effect.

Note: Only stable releases of CodeSnip are displayed in this dialogue box. Any new preview or beta releases are not reported. If you are interested in downloading these, please check the CodeSnip News Feed which carries announcements of all such releases.
