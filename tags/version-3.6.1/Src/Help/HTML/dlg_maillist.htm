Join Mailing List Dialog Box

This dialog box is used to sign up to the CodeSnip mailing list. It is accessed from the Tools | Join Mailing List menu option.

Enter the email address that is to receive mailings in the Email address edit box. Your name or nickname can also be entered in the Name or nickname edit box, but this is optional.

Help links are displayed which can be clicked to learn more about the CodeSnip mailing list or to read the privacy statement.

To proceed with the subscription ensure that you are connected to the internet then click the Subscribe button.

The program will now attempt to connect to www.delphidabbler.com and send the subscription information. Once the subscription request has been processed a message will be displayed that informs whether then request succeeded and how to confirm the subscription if so. Click the Finish button to dismiss the dialog box.

Should an error be detected during the subscription request an error message describing the problem will be displayed.
