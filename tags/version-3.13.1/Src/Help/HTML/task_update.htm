Update Database

Updates to the code snippets database are made available from time to time on the web. You can download the latest version of the database easily by connecting to the internet then clicking the Update Database from Web toolbar button or by choosing the Database | Update From Web menu option.

The Update From Web dialog box will be displayed. Make sure you are connected to the internet then click the Update from Web button to begin the update.

Database updates apply to all users. This is because only one copy of the database exists on the computer.

Updating the database has no effect on any user-defined snippets that have been created – they are not overwritten by the update process.
