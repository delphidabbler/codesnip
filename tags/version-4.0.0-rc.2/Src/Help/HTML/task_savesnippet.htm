Save Snippet to Disk

You can save either a single routine or a whole category of routines to disk as follows.

  1. Select a routine or a category in the Overview Pane.
  2. Click the toolbar button or choose the File | Save Snippet menu option.
  3. Enter the name of the snippet file in the Save Snippet dialogue box.
  4. Choose the file type, the commenting style and whether you want the file syntax highlighted then click OK to save the file.

See the Save Snippet dialogue box help topic for detailed information about the available options.

Only functions, procedures and categories can be saved; type definitions, constants and freeform snippets are not supported.

Note that the default commenting style, syntax highlighting and file type can be configured using the Preferences dialogue box.
