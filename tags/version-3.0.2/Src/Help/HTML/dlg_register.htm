Registration Wizard

This dialog box is used to register CodeSnip.

Note You do not have to register CodeSnip to use it. However, the author hopes you will register so that he knows that CodeSnip is being used and is worth maintaining.

The dialog takes the form of a multi-page wizard that gathers registration information and submits it online. You move through the pages by clicking the Next button and can return to a previous page using the Back button. The Cancel button closes the wizard without registering the application.

First page

This page simply displays some introductory text.

Second page

You use this page to provide information about yourself. As a minimum you need to provide your name or a nickname. You can also provide your email address. However, this is only required if you wish to subscribe to the CodeSnip mailing list. To subscribe you need to check the box below the email address field.

You can click links on the page to learn more about the CodeSnip mailing list or to read the privacy statement.

Third page

Along with the information you have entered CodeSnip will have collected some information that it will send to the web server. All the the information that will be sent is displayed on the page.

If you are happy to send the information click the Submit button which will have replaced the Next button. Note: please ensure you are connected to the internet before clicking the button.

The program will now attempt to connect to www.delphidabbler.com and send the registration information. If the report is sent successfully the wizard's final page will be displayed. Should an error be detected an error message describing the problem will be displayed.

Fourth page

This page is shown when the registration has been successful. The program's registration code will be displayed on the page. This value is automatically stored by CodeSnip in its configuration file.

If you have joined the mailing list a message will be displayed that confirms this. You will also have been sent an email containing a link that you need to click to confirm your registration.

You can now close the dialog box by clicking the Finish button that has replaced the previous page's Submit button.

Please note that each user of the computer where CodeSnip is installed must register the program separately.
