The Main Display

CodeSnip's display is divided into two main parts.

Overview Pane

This is used to list all the snippets in the database. Snippets are organised by category, alphabetically or by snippet kind.

Detail Pane

Used to display detailed information about selected snippets and other items of information. Information is displayed in one or more tabs.

Other Elements

The window also contains a main menu, a toolbar and a status bar where database and search information and help text are displayed.
