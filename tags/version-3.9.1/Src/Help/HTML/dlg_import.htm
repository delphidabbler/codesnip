Import Snippets Dialog Box

This dialog box is used to import user defined snippets from a CodeSnip export files received from other CodeSnip users into the user defined database. The dialog box is opened from the Database | Import Snippets menu option.

The dialog box is a standard windows dialog box. Choose the required import file from it. Start the import by clicking OK. The import can be cancelled by clicking Cancel.

On clicking OK the import process starts. If any of the imported snippets has a name that clashes with a snippet in the existing user database then a subsidiary dialog box is displayed querying whether to overwrite the existing snippet, skip importing the snippet, rename the imported snippet or cancel the whole import. If you choose to rename a snippet the a further dialog box is displayed where the new name can be entered.

When the import completes a message box is displayed giving the names of the imported snippets. If no snippets are imported a message box informs you of this.
