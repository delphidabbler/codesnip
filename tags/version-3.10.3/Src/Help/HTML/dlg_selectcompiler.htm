Select Compiler Dialog Box

This Windows standard dialog box is displayed when the ellipsis button on the Compiler tab of the Configure Compilers dialog box is pressed. Use the dialog box to select the executable file of a command line compiler.

Click OK to confirm the selection. This will close the dialog box and enter the selected file name into the edit box next the the ellipsis button. If the selected file does not exist an error message will be displayed and the dialog box will not close.

Click the Cancel to cancel the selection.
