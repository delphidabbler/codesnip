Main Menu

The main menu contains the following items. Click an item for information about the menu contents.

File Save snippets and units, print snippets and exit the program.
Edit Clipboard and selection actions.
View Navigation, tab selection, tree expand & collapse actions and other information.
Search Search the database by text, compiler or cross-reference. Also allows manual selection of displayed snippets.
Snippets Create, edit and delete user defined snippet. Also allows snippets to be imported or exported and submitted for use in the on-line database.
Categories Create, rename and delete user defined categories.
Database Save, backup and restore the user database and update the main database from the web.
Compile Test compile snippets, view compiler error logs and display the unit used for the test compile.
Tools Configruation, bug reporting and registration.
Help Help file, web links, license, copyright, donations, news and acknowledgements.
